Detailed Description

Menu har hand om menyer innan man börjar spela spelet.

Menu skapar, ritar ut och hanterar input i de menyer spelet har innan spelet startar. Det använder sig av std::string och sf::Text för att skapa listor av val som användaren sedan kan välja med hjälp av piltangenterna och enter.
