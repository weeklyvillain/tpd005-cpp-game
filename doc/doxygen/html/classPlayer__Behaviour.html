Detailed Description

Player_Behaviour ärver från Behaviour och tillhandahåller beteendet för spelare.

Player_Behaviour ärver allting från Behaviour och låter spelaren interagera med spelvärlden. Detta genom att flytta spelare, kollidera med objekt, skjuta projektiler, animera spelaren och flippa spelarens sprite baserat på var spelaren går.
