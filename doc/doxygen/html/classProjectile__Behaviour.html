Detailed Description

Projectile_Behaviour ärver från Behaviour och tillhandahåller beteendet för projektiler.

Projectile_Behaviour ger projektiler deras beteende. Den gör så att projektilen åker ifrån spelaren i den riktining som spelaren tittar. Projektilen åker sedan tills den kolliderar med ett objekt som inte är en projektil eller en spelare. Den dödar objektet om det är en Enemy.
